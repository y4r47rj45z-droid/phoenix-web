Политика использования файлов cookie

Сайт fenix.moscow использует файлы cookie для корректной работы сайта, анализа посещаемости и улучшения пользовательского опыта.

1. Что такое cookie

Cookie — это небольшие текстовые файлы, которые сохраняются на устройстве пользователя при посещении сайта.

2. Для чего используются cookie

  • для обеспечения корректной работы сайта;
  • для сохранения пользовательских настроек, например согласия на использование cookie;
  • для анализа посещаемости сайта с использованием Яндекс.Метрики.

3. Яндекс.Метрика

На сайте может использоваться сервис Яндекс.Метрика, который помогает анализировать посещаемость сайта и улучшать его работу. Данные используются в обезличенном виде.

4. Управление cookie

Пользователь может ограничить или отключить использование cookie в настройках своего браузера. При отключении cookie отдельные функции сайта могут работать некорректно.

5. Контакты

По вопросам использования cookie можно обращаться по e-mail: info@fenix.moscow.
